# RubyJwk
Authenticate JWKs via public JWT enspoint

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ruby_jwk'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ruby_jwk
```
